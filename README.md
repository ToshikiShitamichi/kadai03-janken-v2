# 制限じゃんけんオンライン
## 制作アプリの説明
- [new]制限じゃんけんをオンラインで対人戦ができるようにしました
- 自分が出せる手が制限されたじゃんけん。
- 自分の手札、残りの枚数などの情報から勝てる手を選ぼう。
## 工夫した点・こだわった点
- ホスト側でランダム手札を設定、ゲスト側はDBから手札を取得
- onValueでDBを監視し、双方の手が出揃ってから勝敗判定
- windowでグローバル変数を管理
- ゲーム終了時にホスト側でDBからルームIDを削除
- vminを使用してレスポンシブ対応
## 次回トライしたいこと
- 機能ごとのコミット
- 複数ファイル開発時のリファクタリング
- ラウンジ機能により複数ルームでデッキを共有
